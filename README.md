## Project Summary

This project involved building a Business Intelligence (BI) dashboard in **Microsoft Excel** to analyze performance metrics from an IMDb movie dataset. The goal was to provide a visual summary of the highest-level financial and critical metrics within the dataset.

## Key Insights & Metrics

* **Financial Overview:** Calculated and visualized Gross Revenue ($50M), Average Profit ($50M), and Average Budget ($44M).
* **Director Performance:** Ranked directors by Facebook Likes, with **Steven Spielberg** and **Woody Allen** leading.
* **Critical Metrics:** Identified movies with the highest budget (e.g., 'The Host') and those with the most reviews (e.g., 'Django Unchained').
* **Trend Analysis:** Visualized Sales Trend over time (e.g., 1937 to 1977) and Top Selling Months (e.g., Jan, Feb, Nov).

## Tools & Skills Demonstrated

* **Tools:** Microsoft Excel (Advanced Pivot Tables, Pivot Charts, and Data Visualization).
* **Skills:** Data Aggregation, Financial Analysis, Data Storytelling, Dashboard Design.
